1. OPEN THE PROJECT
   - Open 12th-Engine folder in VS Code
   - Java and LWJGL dependencies are configured in build.gradle.kts

   ```
   .\gradlew build
   ```

   PowerShell:
   ```

   ```

   Level 1: Single target, 2 blocks (Easy)
   - Push the blocks to hit the target sphere
   - Use WASD to move camera, mouse to look around

   - Take your time positioning blocks
   - Watch out for walls

   - Multiple blocks at different heights
   - Complex spatial puzzle
   
   Controls:
   - W/A/S/D: Move camera
   - Mouse: Look around
   - R: Restart level
   - N: Next level (when complete)

рџљЂ HOW TO EXTEND THE GAME

ADD A NEW LEVEL:
1. Open GameManager.java
2. Add a new loadLevel() case:
   ```java
   case 4:
       loadLevel4();
       break;
   ```
3. Create loadLevel4() method with:
   - Ground setup (call setupGround())
   - Target entities
   - Pushable blocks
   - Obstacles

ADD NEW ENTITY TYPE:
1. Create class extending BasicEntity
2. Implement physics/behavior
3. Add rendering logic to Renderer3D

MODIFY PHYSICS:
1. Adjust gravity: Change in BasicEntity constructor
2. Adjust push force: Change in InteractionSystem
3. Modify object properties: Use setMass(), setDrag()
